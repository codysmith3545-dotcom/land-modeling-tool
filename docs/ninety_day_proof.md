# 90-Day Proof Checklist

Use this after running `land-model run`.

## Deliverables

1. **Node map** — `outputs/infrastructure_nodes.json` (ranked by scarcity, capacity, momentum)
2. **Candidate shortlist** — `outputs/top_100_shortlist.json` (100–200 parcels in production)
3. **Diligence memos** — `outputs/diligence_memos/` (10 developer-grade memos)
4. **Backtest** — `outputs/backtest_metrics.json` (leakage-safe labels)
5. **Kill criteria review** — `outputs/ninety_day_proof.md`

## Kill criteria (from config)

- ≥3 actionable parcels from first 10 memos
- ≤70% power-expert rejection on top candidates
- ≥20% of historical winners appear in top 100
- Pivot if >50% of shortlist is already publicly marketed

3. Expand to Ohio/Michigan/Illinois nodes
